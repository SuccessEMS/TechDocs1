Mobile Version of VEMS

VEMS is available both in an version and in a mobile version. If you access VEMS from a mobile device, you are redirected to the mobile version of VEMS. If you access VEMS from a mobile device, you are redirected to the VEMS mobile version.

Icons on the Mobile VEMS Home page

Icon

Description

Click the Home icon at any time to return to the Mobile VEMS Home page.

Click the My Bookings icon to open a page on which you can view all the bookings that you have made in VEMS.

Click the New Bookings icon to open a page on which you can enter bookings in VEMS.

Click the Groups icon to open a page that lists all the groups for which you can make a booking in VEMS. You can also search for groups on this page.

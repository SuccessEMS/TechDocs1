Web User Licensing in VEMS

If your organization has reached your limit of licensed web users, you will no longer be able to create additional web users.  To get back under your maximum number of web users, you can either inactivate or delete web users from the EMS Desktop Client.

Inactivate Web Users

  1. Navigate to Configuration > Web > Web Users. 
  2. Web users can be inactivated by hitting the Edit button on the Web Users screen and then setting the Status field to Inactive.

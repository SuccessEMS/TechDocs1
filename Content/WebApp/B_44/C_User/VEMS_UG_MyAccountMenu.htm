My Account Menu

The My Account menu provides options for editing your user profile and for specifying delegates, which are web users who can assume your identity in VEMS and carry out actions in VEMS on your behalf. It also contains options for options for customizing some default values (Start Time, End Time, and so on) for the different types of room requests and for creating a list of favorite rooms to search for availability.

  • Create or Modify a User Account
  • Create Delegate Accounts
  • Set Default Values for Room Requests
  • Create a List of Favorite Rooms

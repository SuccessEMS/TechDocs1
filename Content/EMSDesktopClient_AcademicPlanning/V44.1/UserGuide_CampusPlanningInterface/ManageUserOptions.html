Manage User Options in the EMS Campus Planning Interface

When your EMS administrator configures your user account, a default password is set for your account. You can customize your own password for the EMS Campus Planning Interface. Your EMS Administrator may have configured custom help text that is displayed on pages in the EMS Campus Planning Interface. You can set the options of the Academic Book or turn off/on the custom help text that is displayed on pages in the EMS Campus Planning Interface.

This section will give you information on doing the following:

  • Change Your EMS Campus Planning Interface Password
  • Set Options for the Academic Book Display
  • Turn On or Off Custom Help Text

Change Your EMS Campus Planning Interface Password

  1. On the EMS Campus Planning Interface menu bar, click Manage User Options. The Manage User Options page opens. The Password Options tab is the active tab.
  2. In the Old Password field, enter your current password for the EMS Campus Planning Interface.
  3. In the New Password field, enter your new password.
  4. In the Confirm New Password field, enter your new password again, exactly as you entered in the New Password field, including spaces and capitalization.
  5. Do one of the following:
    1. Click Save. You remain on the Manage User Options page. An “Options saved successfully” message is displayed at the top of the page. If you changed your password, then the next time you log into the EMS Campus Planning Interface, you must use your new password.
    2. Continue to setting other user options as needed.

Set Options for the Academic Book Display

  1. Open the Book Options tab.
  2. Set the options for the Academic Book display.

Option

Description

Display

The information that is displayed for an event/course in the Academic Book. Options are:

  • Event Name/Course (the default value)
  • Group/Instructor

Start Time

The daily start time for the Academic Book.

Tooltip Options

The items that are to be included in the tooltip that opens when you place the cursor over an event/course in the Academic Book.
  1. Do one of the following:
    • Click Save. You remain on the Manage User Options page. An “Options saved successfully” message is displayed at the top of the page. If you changed your password, then the next time you log into the EMS Campus Planning Interface, you must use your new password.
    • Continue to setting other user options as needed.

Turn on or Off Custom Help Text

  1. Open the Other Options tab. By default, when the Other Options tab first opens, Show Help Text is selected.
  2. Clear Show Help Text to turn off any custom help text that is displayed on any page of the EMS Campus Planning Interface; otherwise, select Show Help Text to turn on any custom help text that is displayed on any page of the EMS Campus Planning Interface. 
  3. Do one of the following:
    1. Click Save. You remain on the Manage User Options page. An “Options saved successfully” message is displayed at the top of the page. If you changed your password, then the next time you log into the EMS Campus Planning Interface, you must use your new password.
    2. Continue to setting other user options as needed.

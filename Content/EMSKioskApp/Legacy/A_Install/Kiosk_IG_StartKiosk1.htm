Launch EMS Kiosk (Legacy)

To launch EMS Kiosk (Legacy):

  1. The first time you access EMS Kiosk, you will be redirected to the Config.aspx page (e.g., http://[ServerName]/EMSKiosk/config.aspx).

    Configuration Page

  2. Enter your server and database information. 
  3. Enter your EMS SQL Server name and Database.  
  4. Click Verify Server to continue. The Kiosk Profile section will appear.

    Kiosk Profile Screen

    Note: Kiosk Profiles are configured within the EMS Desktop Client application. For instructions, Configure EMS Kiosk Profiles (Legacy).

  5. Select a profile for this Kiosk.
  6. Click Save.

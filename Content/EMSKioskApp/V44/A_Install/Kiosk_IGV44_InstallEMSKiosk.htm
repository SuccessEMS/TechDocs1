EMS Kiosk (V44) Installation Guide

This guide provides guidance in administering EMS Kiosk for System Administration and IT users. The EMS Kiosk is an optional module for EMS. You can use the kiosk to view the today’s events, make a reservation for yourself, view your existing reservations, locate a person who has reservations for today, and, if enabled, check yourself into and out of a building.

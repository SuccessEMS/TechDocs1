Create Reservations in EMS for Microsoft Outlook (Legacy)

You can use the functions in the EMS for Microsoft Outlook plugin module to check for available space for an event and to make a reservation for the event that is saved in your EMS database. You can search for rooms that are available for a particular time on one day (a simple reservation with one booking) or on multiple days (a series reservation with multiple bookings).

This section covers the following topics:

  • Create a Single Reservation
  • Create a Series Reservation
  • Create a Video Conference Reservation
  • Edit or Cancel a Scheduled Event

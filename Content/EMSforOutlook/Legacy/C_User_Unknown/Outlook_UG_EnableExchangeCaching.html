Enabling Exchange Caching

Due to performance issues in Exchange, EMS for Outlook Update 7 requires that users use cached mode in their Microsoft® Outlook settings. Users who have disabled Exchange caching will see the following error message when they attempt to run the add-in after installing Update 7. 

  1. In Outlook, click the File tab.
  2. Click Account Settings, and then click Account Settings.
  3. On the E-mail tab, click the Exchange account, and then click Change.
  4. Under Microsoft Exchange server, click the Use Cached Exchange Mode check box.
  5. Exit and then restart Microsoft Outlook.

See Also: Microsoft®'s recommendation: a secure solution for using cached mode.

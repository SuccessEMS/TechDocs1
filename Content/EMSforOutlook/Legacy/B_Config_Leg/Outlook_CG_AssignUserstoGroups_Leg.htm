Assign EMS Users to Groups

You assign users to see groups in EMS by modifying their Everyday User accounts using via the External Reference field.

  1. Navigate to Billing Information tab within Reservations > Reservations > Employees. 

  1. Make a note of the value in the External Reference field.

  1. Navigate to the Everyday User's account, which is on the Everyday User tab within Configuration > Everyday User Applications> Everyday Users.

  1. Open the Everyday User's account by clicking the Edit button. On the Everyday User tab, compare the value above to the value in the  External Reference field.

Note: These values can be set automatically by administrators if your organization is using the HR Toolkit module.  Please see the HR Toolkit Installation Instructions for more information.

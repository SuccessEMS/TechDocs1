Install HR Toolkit for the Groups/Contacts or Hybrid Model

Once you have obtained the installation files, perform the following steps on the master MS SQL Server that hosts your EMS database Using Microsoft SQL Server Management Studio.

  1. Execute the (G_C_GT_WU.sql)/(G_GT_WU.sql)/(Hybrid.sql) script against your master database. This will create the EMS_Staging database containing one table named tblPeople.
  2. Execute the HRTK Update Script (G_C_GT_WU.sql)/(G_GT_WU.sql)/(Hybrid.sql) script against your EMS database.  This will create a stored procedure in your EMS database named HRTK_Update_Group.

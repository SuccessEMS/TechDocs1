Introduction to EMS Plan a Meeting (PAM) Installation Guide

Plan-a-Meeting (PAM) is a component for Virtual EMS that integrates with Microsoft Exchange. With this module, web users can view the availability of both meeting rooms and attendees, and send Microsoft Outlook meeting invitations—all from within Virtual EMS. 

You must be licensed for EMS, Virtual EMS, and PAM in order to configure and use this module. If you are unsure if your organization is licensed for PAM, or if you would like to learn more about it, please contact your Account Executive.

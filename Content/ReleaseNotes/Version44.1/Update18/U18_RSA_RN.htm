EMS Room Sign App Was Not Using Relative Paths

Although assets (images, scripts etc.) were loading correctly from relative URLs, calls to the API appeared to have their URLs constructed incorrectly. As a result, the EMS Room Sign App could not successfully talk to the API.

Fix: Modified functionality so that the EMS Room Sign App uses relative paths.

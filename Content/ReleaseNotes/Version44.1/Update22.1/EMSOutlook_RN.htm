Enhanced Exchange Integration Web Service Logging to Provide Detailed Request Logging for EWS Connectivity Issues

To better debug and troubleshoot Exchange Web Services (EWS) connectivity issues, enhanced EWS request logging was added.

The following logging capabilities were added to Exchange Integration Web Service:

  • Added configuration parameters in the web.config to set the log folder, filename pattern, and retention policy.
  • Support for EWS Managed API request tracing if the log level is set to "Debug."
  • Support for the following log levels as a configuration option in the web.config: Verbose, Debug, Information, Warning, Error, and Fatal.

Internal Reference # 29416

Security

Prevent Cross-Site Scripting Vulnerability

Enabled data sanitization for input fields (server-side) to provide protection from cross-site scripting vulnerability.

Internal Reference # 27168
(EMS Internal Use Only)

Today's Events

Enable Default Sorting on Today's Events Page

Added a parameter (in EMS Kiosk), "Default Sort Column", which enables administrators to set default ascending/descending sort preferences for the Today's Events page from a drop down menu with the following options: Start Time, End Time, Event Name, Employee Name, Location, and Has Map.

Internal Reference # 27746
(EMS Internal Use Only)

Floor Maps

Show/Hide Availability Indicators

Added a parameter (in EMS Kiosk), "'Show Availability Indicators on Locate Space", which enables administrators to show or hide availability indicators on floor map images on kiosks.

Internal Reference # 28464
(EMS Internal Use Only)

Fixes 

Area

Description

Floor Maps

Locate Space Availability Tooltip Misaligned

When users selected Locate Space and clicked on an unavailable space on a floor map, the tooltip indicating whether it was available displayed in the upper left such that it was off-screen.  

Fix: Modified the display so the room availability tooltip displays near the space on the floor map such that it is visible to users.

Internal Reference # 29320
(EMS Internal Use Only)

EMS API Tried to Query EMS_Master Dababase When Running an Update Call

Running update calls on the following APIs produced an error:

  • API_EMS_CreateReservation
  • API_EMS_CreateWebRequest
  • API_EMS_CreateWebUser
  • API_EMS_EditReservation
  • API_EMS_EditWebUser
  • API_EMS_GetWebUserDetails
  • API_EMS_GetWebUsers
  • API_EMS_UpdateWebUserOptions

Fix: Modified functionality so the update runs successfully.

Authentication

 

Unable to Authenticate When Installing or Logging In to EMS Desktop Client Web Deploy Through Proxy Server Environments

The EMS Web Deploy Desktop Client was unable to authenticate through certain proxy servers when users attempted to install or log in.

Fix: Modified functionality so that there is an authentication process in the Web Deploy executable file to pass user authentication to the proxy server.

Fixes  

Area

Description

Floor Plan

Floor Plans Not Loading

When users clicked on floor plans from the View Map column on the Locate Person or My Reservations pages, they did not load successfully.

Fix: Modified functionality so that floor plans load as expected.

24194

Translations

Error Messages in Spanish (Venezuela) and Portuguese (Angola) Incorrect

When translations for EMS Kiosk were set to Spanish (Venezuela) and Portuguese (Angola), an error page titled " Browser Title-Untitled Page" displayed the error, "Details: An unknown error occurred."

Fix: Modified the page to show the correct browser title and error message.

23320

Outlook Reservations

End Now Button Persists for Outlook Reservations

Reservations that had been booked in Outlook incorrectly showed the End Now button.

Fix: Modified validations so that the End Now button does not show on Outlook reservations. 

23966

Check-In

Reduce Check-In to Single Click

Users had to confirm check-in with a second click.

Fix: Modified the check-in confirmation to only require one click on the Check-In icon.

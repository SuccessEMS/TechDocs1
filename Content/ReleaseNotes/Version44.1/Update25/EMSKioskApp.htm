iFrame

Badge Info Was Not Input into Badge Field When EMS Kiosk App Was Displayed in iFrame

When the EMS Kiosk App was displayed in an iFrame software display, the user's badge info is not being input into the badge field.

Fix: Modified functionality so that badge info is passed to the badge field input and validation is initiated.

Internal Reference # 35531
(For EMS Internal Use Only)

Provisioning

 

EMS Kiosk App Lost Provisioning After Session Expiration

After the EMS Kiosk App session expired, the Kiosk needed to be provisioned.

Fix: Modified functionality so that the EMS Kiosk App provisioning survives token expiration.

Internal Reference # 37125
(For EMS Internal Use Only)

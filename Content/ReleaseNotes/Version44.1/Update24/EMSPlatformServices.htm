Autodiscover

 

Fallback for Discovery

If auto-discovery fails with the one Skype for Business Autodiscover URL, there is now a collection of Skype for Business discovery URLs that can be configured as a fallback.

Internal Reference # 37565

Skype for Business Configuration Change

Skype for Business configuration has been moved from the EMS Desktop Client to the Admin User Interface (UI).

Internal Reference # 37649
(For EMS Internal Use Only)
 

Users Able to Search Bookings by Multiple Reservations

Users can filter their search by multiple Reservation IDs.

EMS Internal Reference # 37548
(For EMS Internal Use Only)

Fix

Area

Description
 

Bookings and Reservations Were Created Using Invalid Group ID

Users could create bookings and reservations using invalid data (e.g., Group ID).

Fix: Modified functionality so that reservations and bookings cannot be created if invalid data was passed in the create requests.

Area

Description

Subscription Function

Booking Date/Time Changes Were Not Showing in Calendars Updated through Subscription Function

For users running Master Calendar connector to sync events from EMS to Master Calendar, booking date/time changes made in EMS were not showing in calendars updated through the subscription function.

Fix: Modified functionality so that booking date/time changes made in EMS will appear in calendars updated through the subscription function.

Internal Reference # 36022
(For EMS Internal Use Only)

Security

EMS Master Calendar Security Vulnerability Addressed

Fix: Modified functionality to address a security vulnerability in EMS Master Calendar.

Internal Reference # 36985
(For EMS Internal Use Only)

List View Expand/Collapse Event Button

 

EMS Master Calendar ADA Compliance Issue—Tabbing Not Functioning Properly

EMS Master Calendar had an ADA compliance issue because tabbing was not functioning properly in list view, and users were unable to expand/collapse content on the Master Calendar site.

Fix: Modified functionality so that tabbing functions properly in EMS Master Calendar, and there is no ADA compliance issue.

Internal Reference # 37211
(For EMS Internal Use Only)

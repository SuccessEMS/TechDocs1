Fixes  

Area

Description

Schedule View

Unable to Remove Room from Schedule View or Remove Room from Selected Room

When selecting a room for a booking in the Schedule view, unchecking a room was not removing that room from the Schedule view.

Fix: Modified functionality so that rooms that are unchecked are removed from the Schedule view.

Internal Reference # 32114

Services Tab

Message Appeared Despite Service Order Already Being Applied

When applying a resource, users were receiving the error message, "You have an unapplied service order. Are you certain you wish to continue?" despite the service order having already been applied.

Fix: Modified functionality so that the error message does not appear for previously applied service orders.

Internal Reference # 31819
